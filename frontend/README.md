# EventHub Frontend

This is the Next.js frontend for the EventHub SaaS platform.

## Tech Stack
- Next.js (TypeScript)
- Tailwind CSS
- Apollo Client (GraphQL)
- JWT Auth (placeholder)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env.local` and update values.
3. Run the development server:
   ```bash
   npm run dev
   ```

## Environment Variables
See `.env.example` for required variables.
